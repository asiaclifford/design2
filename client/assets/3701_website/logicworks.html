Libraries
  • LogicWorks HomePage
  • LogicWorks patch to latest version for 4.0 from LogicWorks.
  • When you save the below library files, be sure that they still have the .clf extension. If not, just change the extension to .clf and everything should then work.
  • PALS_EMS.clf: PAL16V8 and PAL/GAL 22V10 Library File (pinouts only)
  • PR_GA_PL_EMS.clf: Primary Gates PLUS Library (includes mixed-logic gates not in primgate.clf)
  • PR_IO_PL_EMS.clf: Primary IO PLUS Library (includes mixed-logic input switches not in primio.clf)
  • 74_PLUS_EMS.clf: Additional 7400 Devices (not in 7400dev.clf)
  • mem_EMS.clf, includes memory devices (including EEPROM 2864 and 62256 32kx8 SRAM)
  • 68HC11E9_EMS.clf: 68HC11 E9 Library File (pinout only)
  • For LogicWorks version 3 only: pr_ga_pl.clf
  • For LogicWorks version 3 only: 74_plus.clf
  • For LogicWorks version 3 only: pr_io.clf
  • Pinouts.cct: LogicWorks file for many chips.
